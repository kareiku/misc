# Miscellaneous

A collection of small projects, guides and scripts I keep handy—for myself and anyone who might find them useful.

## Table of Contents

- Various scripts for a wide range of purposes
- **DBM4J**: A lightweight set of Java classes and interfaces for easy database access. Designed for simple queries, yet fully extensible.
- **LM4J**: Simplifying natural language handling in Java. These classes make it easy to read lang-like (plain text) files and return results safely using utilities like `Optional`.

## Features

- Modular components, reusable across many projects
- Lightweight and extensible design philosophy

## Usage

- Scripts are organized in directories named after the programming language they're written in.
- Precompiled versions of libraries and applications are available in their respective `target/` directories.

## Build It Yourself

1. Clone the repository.
2. Navigate to the directory of the project you want to compile.
3. Compile:
   - For projects that use a build tool (Maven, Gradle, CMake, etc.), use the standard commands associated with that tool.
